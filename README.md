# sw1pr0g's personal blog

[See it live!](https://blog.sw1pr0g.com)


## Development


Make sure you have `ruby 3.3.10` installed, use `rbenv` in case you don't.


```bash
rbenv install
```

Then run:

1. `bundle install`
2. `bundle exec jekyll serve --safe --strict_front_matter`
3. Done!
